Control bit is NX = 1, f = 1, no = 1, X = 9 (0b0000 0000 0000 0000 0000 1001) and Y = 5 (0b0000 0000 0000 0000 0000 0101). If NX = 1, it is inverted to X to -9 (0b1111 1111 1111 1111 1111 0111). When f = 1, the output is X + Y, it is (0b1111 1111 1111 1111 1111 0111) + (0b0000 0000 0000 0000 0000 0101) = (0b1111 1111 1111 1111 1111 1100), in decimal is (-9) + (5) = (-4). no is invert to output, which changes (0b1111 1111 1111 1111 1111 1100) to (0b0000 0000 0000 0000 0000 0100), in decimal is 4. In other words, this control flag means X - Y; for example, 9 - 5 = 4.
